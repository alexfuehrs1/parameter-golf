# Skill: Hypothesis-Driven Ablations

## Purpose
Ensure experiments produce interpretable evidence.

## Instructions

Before each experiment, write:
- the single main change
- the expected mechanism
- the expected metric movement
- the alternative explanation if it fails

Prefer changing one factor at a time.
If multiple changes are bundled, explicitly state that attribution becomes weak.

## Rules

- No ablations without a hypothesis
- No broad “optimization bundles” unless deliberately exploring interaction effects
- Distinguish signal from noise
