# Skill: Verify Before Finish

## Purpose
Ensure correctness before declaring completion.

---

## Checklist

Before finishing:

- Relevant tests pass
- No regressions introduced
- Lint passes (if applicable)
- Build passes (if applicable)
- Only intended files modified
- Scope respected

---

## Rules

- Do not claim completion prematurely
- If unsure, state uncertainty
- Always list remaining risks
